## Development

1. Install NodeJS.
1. Install yarn

   ```bash
   npm i -g yarn
   ```

1. Install node modules

   ```bash
   yarn
   ```

1. Start project

   ```bash
   yarn dev
   ```

## Build project

1. Install NodeJS.

1. Install yarn

   ```bash
   npm i -g yarn
   ```

1. Install node modules

   ```bash
   yarn
   ```

1. Build project

   ```bash
   yarn build
   ```

1. Deploy to gh-pages

   ```bash
   npx gh-pages -d dist -b gh-pages
   ```
